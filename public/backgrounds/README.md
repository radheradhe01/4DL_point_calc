# Background Templates Directory

Place your background template images in this directory.

## File Structure

```
public/backgrounds/
├── template1.jpg              # Full resolution image
├── template1-preview.jpg      # Preview image (smaller)
├── template2.jpg
├── template2-preview.jpg
├── template3.jpg
├── template3-preview.jpg
├── template4.jpg
├── template4-preview.jpg
├── template5.jpg
└── template5-preview.jpg
```

## Image Specifications

- **Full Images**: 1920x1080px or higher, optimized JPG/PNG
- **Preview Images**: 400x225px, optimized JPG (under 100KB)

See `BACKGROUND_TEMPLATES_SETUP.md` for detailed instructions.
